# Quick start for tomorrow

1. Open `presentation/optikuh_eda_presentation.pdf` and present from it.
2. Read `presentation/speaker_notes.md` once before presenting.
3. In VS Code, open this folder and run `python src/02_run_eda.py` only if you need to regenerate outputs.
4. Keep the message simple: this is EDA only; no model is trained.

Main numbers to remember:
- 492,540 daily records
- 1,714 animals
- 12 anonymized farms
- approx. 3,422 lactation episodes
- health status is lactation-level, while the main table is daily-row level
- biomarker missingness is expected because samples were taken at selected time points
